# Enterprise-Side · AI教学知识库平台 — 企业招聘端

> 面向酒店管理行业的AI驱动企业招聘平台，帮助企业精准匹配学生人才、管理招聘全流程。

## 📐 功能架构

### 工作台
| 模块 | 功能 |
|------|------|
| **数据概览** | 在库学生总数、月度投递量、在招职位数、录用人数四大核心指标，含环比趋势 |
| **投递趋势** | 近6个月柱状图，直观呈现招聘周期波动 |
| **职位分布** | 各职位投递量进度条对比 |
| **评分分布** | 人才综合评分分段柱状图 + 平均分/优秀率统计 |

### 人才库
| 模块 | 功能 |
|------|------|
| **学生画像库** | 卡片/列表双视图切换，按院校、学历、评分多维筛选 |
| **学生画像抽屉** | 右侧滑出详情面板：6维能力雷达（服务礼仪/前台接待/投诉处理/英语沟通/餐饮服务/应急处理）、综合评分、课程完成率、答题成绩、实训次数 |
| **AI 匹配度** | 基于学生能力与职位需求的智能匹配，三档标签：🟢 高匹配 ≥90% / 🔵 中匹配 75-89% / 🟠 低匹配 <75% |
| **批量操作** | 多选学生 → 批量邀请投递 / 批量对比 |
| **收藏人才** | 一键收藏，收藏列表独立页面管理，侧边栏实时计数徽章 |

### 院校数据
| 模块 | 功能 |
|------|------|
| **合作院校列表** | 6所院校（浙江旅游/上海旅专/北京联合/广州番禺/成都职院等），左侧列表切换 |
| **院校详情** | 右侧面板动态展示：学生数、平均分、活跃率、5项能力维度进度条、5项行为数据 |
| **层次标签** | 本科 / 专科分类，酒店管理 / 旅游管理专业标注 |

### 招聘管理
| 模块 | 功能 |
|------|------|
| **职位管理** | 发布/编辑/关闭/重新开放职位，5个示例岗位（前台接待/客房服务/餐饮运营/大堂副理/宴会策划） |
| **投递管理** | 全流程状态流转（待处理 → 面试中 → 已录用/已拒绝），支持按职位/状态筛选，Excel导出 |
| **面试安排** | 周视图日历 + 新面试表单（日期/时间/方式/地点/面试官），面试评价（星级 + 标签 + 文字） |

### 系统功能
| 模块 | 功能 |
|------|------|
| **全局搜索** | 顶栏搜索框，支持学生/院校/职位三类实体模糊匹配，搜索结果分组下拉 |
| **消息通知** | 新投递/人才推荐/面试提醒/录用成功/职位到期，可点击跳转，一键全部已读 |
| **企业设置** | 企业信息编辑、HR账号管理（管理员/普通成员）、会员套餐（旗舰版） |

## 🛠 技术栈

| 层面 | 选型 |
|------|------|
| 框架 | 原生 HTML / CSS / JavaScript（零依赖） |
| 字体 | Inter + Noto Sans SC（Google Fonts） |
| 布局 | Flexbox + CSS Grid，桌面端优先 |
| 主题 | CSS 变量体系（`--accent` / `--bg-primary` / `--text-primary` 等） |
| 交互 | 事件委托 + DOM 操作，页面切换 fadeIn 动画 |
| 构建 | 无需构建，单文件部署 |

## 🎨 设计规范

### 色彩体系

| 变量 | 色值 | 用途 |
|------|------|------|
| `--accent` | `#3B7EF8` | 主色 / 强调 / 按钮 |
| `--success` | `#00C48C` | 成功 / 在线 / 录用 |
| `--warning` | `#FF8C42` | 警告 / 待处理 / 面试中 |
| `--danger` | `#FF4757` | 危险 / 未读 / 错误 |
| `--purple` | `#6C5CE7` | 紫色标签 / 管理员 |
| `--bg-primary` | `#F0F2F7` | 页面背景 |
| `--text-primary` | `#1a2035` | 主文字（也用于侧边栏背景） |

### 字体

```css
--font-sans: 'Inter', 'Noto Sans SC', 'PingFang SC', 'Microsoft YaHei', system-ui, sans-serif;
--font-display: 'Inter', 'Noto Sans SC', 'PingFang SC', sans-serif;
```

### 关键组件

- **卡片圆角**：14px，投影 `0 1px 4px rgba(0,0,0,.06)`
- **按钮圆角**：8px，主色渐变投影
- **标签**：6px 圆角，语义化配色（蓝/绿/橙/红/紫/灰/金）
- **侧边栏**：220px 宽，可折叠至 64px，深色背景 `#1a2035`
- **进度条**：6px 高，渐变填充（蓝/绿/橙）

## 🔗 与其他端的配合

本平台是 **AI教学知识库平台** 的企业端，与以下端协同：

| 端 | 仓库 | 定位 |
|----|------|------|
| 👨‍🏫 **教师端** | [teacher-ai-portal](https://github.com/haining3140706-hue/teacher-ai-portal) | 课程管理、学生管理、数据看板、题库管理 |
| 📱 **学生端** | [hotel-training-app](https://github.com/haining3140706-hue/hotel-training-app) | 课程学习、AI角色训练、三导师问答、学习报告 |
| 🏢 **企业端** | **Enterprise-Side**（本仓库） | 人才画像、职位发布、投递管理、面试安排 |

数据流转关系：**学生端学习数据 → 教师端管理评估 → 企业端精准匹配招聘**
